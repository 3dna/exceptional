Install Exceptional in your rack app

1. Install the exceptional gem:

{% highlight bash %} $ gem install exceptional {% endhighlight %}

2. Generate an Exceptional config file (config/exceptional.yml)

{% highlight ruby %} $ exceptional install YOUR-API-KEY {% endhighlight %}

You will find your api key in your apps settings in Exceptional.

3. Configure your config.ru file:

{% highlight ruby %} require 'rack/exceptional' use Rack::Exceptional {% endhighlight %}

4. Test everything is working

{% highlight bash %} $ exceptional test {% endhighlight %}

You will see a test exception in your exceptional app screen

5. Enabling Exceptional for non production rack environments

By default Exceptional is only enabled for the Production environment. If you want exceptional collecting errors for other environments then enable that environment in the config/exceptional.yml file. e.g {% highlight bash %} development enabled: true {% endhighlight %}

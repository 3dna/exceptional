Install Exceptional in your Rails app

1. Install the exceptional gem:

{% highlight bash %} $ gem install exceptional {% endhighlight %}

2. Configure your environment.rb file:

{% highlight bash %} config.gem 'exceptional', :version => '2.0.6' {% endhighlight %}

3. Configure Exceptional for your app

{% highlight bash %} $ exceptional install YOUR-API-KEY {% endhighlight %}

You will find your api key in your apps settings in Exceptional.

4. Test everything is working

{% highlight bash %} $ exceptional test {% endhighlight %}

You will see a test exception in your exceptional app

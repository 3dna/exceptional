Install Exceptional in your rails app

1. Install the exceptional gem:

	$ gem install exceptional

2. Configure your environment.rb file:

	config.gem 'exceptional', :version => '2.0.6'

3. Generate the Exceptional config file (config/exceptional.yml)

	$ exceptional install YOUR-API-KEY

You will find your api key in the apps settings screen in Exceptional.

4. Test everything is working

	$ exceptional test

You will see a test exception in your exceptional app screen

5. Enabling Exceptional for non production rails environments

By default Exceptional will only collect errors and exceptions for the production rails environment (as specified by RAILS_ENV). If you want exceptional collecting errors in other environments then you must enable that environment by adding the following lines to the config/exceptional.yml file.

e.g for the development environment

	development:
	  enabled: true

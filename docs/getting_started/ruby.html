Install Exceptional in your ruby apps

1. Install the exceptional gem:

$ gem install exceptional

2. Generate an Exceptional config file (config/exceptional.yml)

{% highlight ruby %} $ exceptional install YOUR-API-KEY {% endhighlight %}

You will find your api key in your apps settings in Exceptional.

3. Enable Exceptional in your ruby code

{% highlight ruby %} require 'rubygems' require 'exceptional' {% endhighlight %}

4. Use the Exceptional.rescue block to catch exceptions

{% highlight ruby %} Exceptional.rescue do # Add your Ruby code here! end {% endhighlight %}

This block will report exceptions to exceptional and re-raise them

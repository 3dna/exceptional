Install Exceptional in your Heroku Rails app

Use the Heroku command line tool to add the exceptional add-on to your rails app

$ heroku addons:add exceptional --app APP_NAME

Where APP_NAME is your heroku application name

You will now be able to click through directly from your heroku app dashboard through to your exceptional errors list!

Check out the heroku docs site for further details about heroku deployment

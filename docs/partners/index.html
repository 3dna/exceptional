adsdsaPartner API

Authentication

All requests should be made to https://www.getexceptional.com using HTTP Basic Auth

Creating an app on exceptional

POST /partners/apps

Request parameters

partner_app_id
something like app123@heroku.com
name
Name of app as it will appear in exceptional (eg stormy-mountain-34)
owner
email address of the user who 'owns' the app, at the moment you must be the 'owner' to do certain things, like configuring 3rd party services.

Response (JSON)

When successful (201 created)

{% highlight ruby %} def something end {:api_key => [API_KEY used to configure gem/plugin], :id => [ID used to identify the app with exceptional], :user_id => [ID used to identify the user who owns this app]} {% endhighlight %}

When failure (422 unprocessible entity)

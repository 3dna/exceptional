Sending extra data to exceptional

1. Use Exceptional.context to set the extra data you want to send (for rails this could be in a before_filter):

	Exceptional.context(:user_id => "1", :email => "someuser@mydomain.com" )

3. When an exception is thrown the extra data is visible in the params section of the exception page

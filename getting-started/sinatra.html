Install Exceptional in your Sinatra app

1. Install the exceptional gem:

	$ gem install exceptional

2. Add Exceptional to your sinatra app

	require 'exceptional'
	use Rack::Exceptional, API_KEY

You will find the API_KEY in the apps settings screen within Exceptional.

3. Ensure :raise_errors is set to true

	set :raise_errors, true

4. Ensure Exceptional for Rack is being loaded

Check 

log/exceptional.log
log file

Example

require 'rubygems'
require 'sinatra'

set :raise_errors, true

require 'exceptional'
use Rack::Exceptional, API_KEY

get '/' do
  "Hello World!"
end

get '/error' do
  big.problem!
end

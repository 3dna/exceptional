Integrating with 3rd Party Services.

Pivotal Tracker:

We require you to have a Pivotal Tracker account. The quick summary is:

  • You can pull errors from Exceptional into a PT project as bugs utilizing PT's "Integrations" feature
  • Updates from PT can be propagated back to Exceptional utilizing PT's "Activity Web Hook" feature

Instructions for use:

Adding exceptions as "bug" stories in PT
  • From your PT Project page, click on "Project" --> "Configure Integrations"
  • Under the "External Tool Integrations", from the pull-down select on the right choose "Other"
  • Fill in the form with these values:
    Name: Exceptional
    Basic Auth Username: your exceptional username
    Base URL: https://www.getexceptional.com/exceptions/
    Import API URL:  https://www.getexceptional.com/apps/
    • {app ID}/exceptions.xml?template=pivotal
  • Click save
  • Back on your PT Project page, click on "More" --> "Exceptional" to view new exceptions in their own panel
Getting updates from PT in your Exceptional errors
  • Again, from you PT Project page, click on "Project" --> "Configure Integrations"
  • Under the "Activity Web Hook" enter these values:
    Web Hook URL: https://www.getexceptional.com/apps/"app ID"/pivotal_activities?api_key="YOUR-API-KEY"
    API Version: v3.
  • Click Save
